# check-idade

Projeto simples feito em Python para treinar lógica de programação.

Usei o ChatGPT para criar o desafio e tentei resolver sozinho.

## O que o projeto faz
- Verifica a idade do usuário
- Libera ou bloqueia o acesso
- Mostra loading no terminal
- Limpa a tela automaticamente

## Bibliotecas usadas
- os
- time
- tqdm
